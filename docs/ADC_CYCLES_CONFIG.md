## Background

The STM32G031 ADC requires a minimum sampling time for the internal sampling
capacitor to charge to within 0.5 LSB of the input voltage. The required time
depends on source impedance:

```
t_SAMPLE ≥ (R_ADC + R_SOURCE) × C_ADC × ln(2^(N+2))
```

**G031 constants** (from RM0444 / DS12994):
- `R_ADC` ≈ 6 kΩ (internal switch resistance)
- `C_ADC` ≈ 7 pF (sampling capacitor)
- `N` = 12 (ADC resolution in bits)
- `ln(2^14)` ≈ 9.70

**ADC clock**: PCLK/2 = 64 MHz/2 = **32 MHz** → 1 cycle = 31.25 ns

## Source Impedance vs. Minimum Sample Time

| Sample time          | Duration @ 32 MHz | Max R_SOURCE |
|----------------------|-------------------|--------------|
| 1.5 cycles           | 47 ns             | insufficient for any source |
| 7.5 cycles           | 234 ns            | insufficient |
| 12.5 cycles          | 391 ns            | ~0 Ω         |
| 19.5 cycles          | 609 ns            | ~3 kΩ        |
| **39.5 cycles**      | **1234 ns**       | **~12 kΩ**   |
| 79.5 cycles          | 2484 ns           | ~31 kΩ       |

## Current Setting

`ADC_SAMPLETIME_39CYCLES_5` — chosen to cover direct encoder outputs and
typical RC anti-alias filter resistors (e.g. 4.7 kΩ series + shunt cap).

With 4× oversampling this yields a total conversion time of:

```
4 × (39.5 + 12.5) = 208 ADC cycles ≈ 6.5 µs per channel
2 channels × 6.5 µs ≈ 13 µs per DMA batch
```

## Recalculating for a Different Source Impedance

Solve for the minimum number of ADC cycles:

```
t_SAMPLE = (R_ADC + R_SOURCE) × C_ADC × ln(2^14)
         = (6000 + R_SOURCE) × 7e-12 × 9.70

cycles = ceil(t_SAMPLE × f_ADC) + 0.5   (round up to next available setting)
```

Then pick the next available `ADC_SAMPLETIME_*` value above that cycle count.
If source impedance is unknown, measure or bound it from the schematic
(series resistors, output driver spec of the encoder IC).
